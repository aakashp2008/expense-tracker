# Expense Tracker

## 📌 About the Project

The Expense Tracker is a simple Python application that helps users record and manage their daily expenses through a menu-driven interface.

## ✨ Features

* Add a new expense
* View all recorded expenses
* Calculate the total expense
* Exit the application

## 🛠️ Technologies Used

* Python

## 🎯 Learning Outcomes

This project helped me learn:

* Python lists
* Loops and conditional statements
* Menu-driven programming
* Basic financial data management
